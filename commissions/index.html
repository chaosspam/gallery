Order Info

Please DM me on Twitter to place an order! Please include the following info:
  • Preferred name if not Twitter name
  • Type of commission
  • Character you want me to draw
  • Where the character's from (probably not necessary if Cygames)
  • Descriptions (how I should draw them: context, emotion, backgrond, clothes)
  • Dimensions (Portrait/Landscape/Square, Base: 2048px × 3640px*, about this big)
  • Reference images of your character
  • Other reference images for poses
  • If you want me to publish the work

* (Mostly a device limitation, my iPad don't like it when the canvas is too beeg :( )

If a reference for the character is not provided, I will not be able to add details of the character that I might have missed after sketch is approved.

If a pose reference for the character is not provided, I will not be able to change the pose of the character after sketch is approved.

By sending me an order you confirm that you have read and accept the terms of service.
