Order Process

Ordering

Please DM me on Twitter to place an order! Include the following info:
Order Info
  • Preferred name if not Twitter name
  • Type of commission
  • Character you want me to draw
  • Where the character's from
    • Game/Anime/OC... probably not necessary if Cygames
  • Descriptions
    • How I should draw them: context, emotion, backgrond, clothes
  • Dimensions
    • Portrait/Landscape/Square, 2048px × 3640px / 2048px × 2048px
    • About this big
    • Size is limited by device, my iPad don't like it when the canvas is too beeg :(
    • Pixel art max size is 128px × 128px, anything larger may cost more
  • Reference images of your character
    • If a reference for the character is not provided, I will not be able to add details of the character that I might have missed after sketch is approved.
  • Other reference images for poses
    • If a pose reference for the character is not provided, I will not be able to change the pose of the character after sketch is approved.
  • If you want me to publish the work

By sending me an order you confirm that you have read and accept the terms of service.

Payment

After the commission is accepted, 50% payment will be requested upfront (can also pay 100% upfront). I will start working on your piece after the payment is processed.

Available payment methods are:

  • Ko-fi

Your commission may be placed at a lower priority or cancelled if the payment is not recieved within 72 hours of the order.
